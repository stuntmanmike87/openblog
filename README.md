# Le projet

Open**Blog** est à l'origine un **blog multi auteurs** créé avec **Symfony 7** ...

## Technologies

Open**Blog** sera développé en utilisant :

- Docker
- Symfony 7
- Mysql 8.0
- PHP 8.3
- PHPMyAdmin
- Sass
